# Library Book Management System 📚

## 📌 Overview  
A Spring Boot web application to manage books in a library.

## 🚀 Live Demo  
[Click Here to View](https://library-book-app.onrender.com)  

## 🛠️ Technologies Used  
- Java (Spring Boot)
- Spring Data JPA (PostgreSQL/MySQL)
- Thymeleaf
- Bootstrap (for UI)
